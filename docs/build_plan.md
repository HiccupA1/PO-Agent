# Build Plan

## Phase 0: Scaffold and Mock Agent

Create the local full-stack scaffold, deterministic agent workflow, mock tools, trace output, and sample data.

## Phase 1: Acceptance Criteria Drafting

Improve acceptance criteria quality, add templates, and support multiple story types.

## Phase 2: Epic Decomposition and DoR Checks

Expand epic slicing heuristics and make Definition of Ready checks more structured.

## Phase 3: Backlog Scoring

Add scoring for value, risk, urgency, dependency load, and readiness.

## Phase 4: Real MCP Integrations

Integrate Jira MCP, Microsoft Graph MCP, Teams, SharePoint, and persistent audit logging.

## Phase 5: Claude Agent SDK / Production-Grade Agent Loop

Replace deterministic mock behavior with a production-grade agent loop, stronger observability, review checkpoints, and deployment-ready controls.
